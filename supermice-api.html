MiceRaid API

MiceRaid API ile istediğiniz üyenin first,peynir,tur,kayıt tarihi gibi bilgilerine JSON formatında erişebilirsiniz.

username = Kullanıcı adı

priv = Yetki (10=Admin,8=Supermod,3=VIP,1=Normal)

first = Birinci olarak toplanan peynir

peynir = Toplanan peynir

save = Şaman olarak kurtarılan fare

title = Ünvan kodu

title_name = Ünvan ismi

round = Oynanan toplam tur

reg = Kayıt tarihi

Örnek:

{ “username”: “Tufan”, “priv”: “10″, “first”: “1180″, “peynir”: “3773″, “save”: “981″, “title”: “281″, “title_name”: “Yaşayan Efsane”, “round”: “47526″, “reg”: “1393167602004″ }

 

MiceRaid API, her 24 saatte 1 kez yenilenir.

MiceRaid API’yi kullanmaya başlayın!

www.supermice.net/api/Kullanıcı_adı

Örnek:

www.supermice.net/api/Tufan

Güncel sürüm: v1.1
